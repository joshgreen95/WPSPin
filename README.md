Forked from the other repo listed on HTB. This one works.

This is an updated version — the original package install is no longer required. Everything is now a single standalone Python script (`wpspin.py`) that runs directly with `python3`.

Also includes `pinguess.sh`, a bash script that uses `wpspin` to generate candidate PINs for a target MAC and feeds them to `reaver` one by one.
